# mongoPlus

#### Description
mongo框架，旨在减少mongoTemplate查询时各种Query的编写，较少各种重复劳动，本框架使用jdk动态代理，类似mybatis框架，用户只要使用接口+注解就可以实现简单的增删改查，对于复杂的查询等操作，可支持输入Query或Criteria查询条件
